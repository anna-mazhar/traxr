# traxr

Point **your own agent** at **your own data**, run controlled-perturbation
experiments, and get back contamination/divergence metrics (`d_norm`, `t*`,
manifestation, token overhead).

Operationalizes the paper *"Trace-Level Analysis of Information Contamination
in Multi-Agent Systems"* (Mazhar, Suri, Galhotra).

> **Status: pre-release scaffolding (`0.1.0.dev0`).** The full README — with
